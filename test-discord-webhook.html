Troubleshooting

Common Issues:

Invalid Webhook URL

Webhook URLs must start with https://discord.com/api/webhooks/

404 Not Found

The webhook may have been deleted. Create a new one in Discord.

Unauthorized

Check that you copied the complete URL including the token at the end.

CORS Error

This is normal for Discord webhooks. The message was still sent successfully!

Rate Limited

Discord limits webhooks to 30 requests per minute. Wait a moment and try again.
