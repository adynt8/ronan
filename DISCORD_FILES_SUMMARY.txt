================================================================================
CONFIGURATION REQUIRED
================================================================================

1. Create Discord Application
   → https://discord.com/developers/applications

2. Get Client ID and Secret
   → OAuth2 section of Discord application

3. Configure Redirect URI
   → https://yourdomain.com/discord-callback.html

4. Update scripts/discordOAuth.js
   → Line 3: clientId
   → Line 93: client_secret

5. Create Discord Webhook
   → Server Settings → Integrations → Webhooks

TESTING WORKFLOW

4. Sync: Run calculation and click "Sync Results"
