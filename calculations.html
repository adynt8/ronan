PlsSize.Me - Calculations

This page explains the calculations used in the PlsSize.Me calculator.

Percentile Calculation

Percentile = 100 * (0.5 * (1 + erf((value - mean) / (standardDeviation * √2))))

Where erf is the error function, which is approximated using a polynomial expansion.

Nominal Width Calculation

Nominal Width (mm) = Erect Girth * 0.45 (for cm) or Erect Girth * 11.43 (for inches)

Volume Calculation

Volume = π * (Erect Girth / (2π))² * Erect Length - 30 (for cm) or (π * (Erect Girth / (2π))² * Erect Length * 16.3871 / 29.5735) - 1 (for inches)

This assumes a cylindrical shape. The result is in ml for cm or fl oz for inches, with adjustments for average internal volume.

Surface Area Calculation

Surface Area = 2π * (Erect Girth / (2π)) * (Erect Girth / (2π) + Erect Length)

This assumes a cylindrical shape. The result is in cm² or in².

Size Distribution Chart Calculations

The chart uses a normal distribution for both length and girth:

Probability = exp(-0.5 * ((x - mean) / stdDev)²) / (stdDev * √(2π))

Data points are generated by combining probabilities for length and girth.
